## 📋 About

This repository contains individual and team-based lab practices exploring core concepts of distributed and parallel computing. Each practice builds on the previous one, progressing from basic thread creation to a full distributed system monitor with a Swing GUI.

---

## 📁 Contents

| # | Folder | Description |
|---|--------|-------------|
| 01 | **Practica-1** | Random number generation and multithreaded average calculation using Java threads |
| 02 | **Practica-2** | Threaded vs. non-threaded average calculation comparison; performance graphing with execution time metrics |
| 03 | **Practica-3** | Client-server communication using Java Sockets |
| 04 | **Practica-4** | Team assignment — distributed computing task (see PDF for details) |
| 05 | **Practica-5** | Dynamic system monitoring — real-time resource tracking |
| 06 | **Practica-6** | Distributed system monitor with Swing GUI — server collects CPU, memory, disk, and network metrics from connected clients |
| 07 | **Practica-7** | Team assignment — distributed computing task (see PDF for details) |
| 08 | **Practica-8** | Team assignment — distributed computing task (see PDF for details) |

> Each folder contains an `EQ/` subfolder with the team version and the individual version where applicable, along with the corresponding practice PDF.
