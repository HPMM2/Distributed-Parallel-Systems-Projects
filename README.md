## 📋 About

This repository contains lab practices and a final integrating project (PIA) that progressively build distributed and parallel computing skills in Java. Topics covered include thread creation and management, parallel performance measurement, TCP/UDP socket communication, real-time resource monitoring, and dynamic role assignment in distributed networks.

---

## 📁 Contents

| # | Folder | Description |
|---|--------|-------------|
| 01 | **Practica-1** | Generates random numbers and calculates their average using multiple Java threads with an `ExecutorService` and `Callable` tasks |
| 02 | **Practica-2** | Compares threaded vs. non-threaded execution time for average calculation and plots performance results as a graph |
| 03 | **Practica-3** | TCP client-server communication using Java Sockets |
| 04 | **Practica-4** | Parallel computing practice |
| 05 | **Practica-5** | Dynamic real-time system resource monitoring |
| 06 | **Practica-6** | Distributed monitor with a Swing GUI — server collects CPU usage, memory, disk, and network metrics from multiple connected clients simultaneously |
| 07 | **Practica-7** | Distributed systems practice |
| 08 | **Practica-8** | Distributed systems practice |
| 09 | **PIA** | Final project — distributed network of nodes with dynamic server/client role assignment, UDP broadcast for role transfer, TCP connections for real-time resource reporting, and a Swing dashboard showing all connected nodes |

> Each folder contains an `EQ/` subfolder with the team version, plus the individual version where applicable.
